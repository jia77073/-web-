📐 数学小课堂 · 中间有0的乘法
🧠 核心规则 三年级必学
多位数（中间有0）乘一位数，计算时每一位都要乘！
0 乘任何数都得 0，但要注意进位哦～
🌰 例题   2 0 3 × 4 = ?
❶ 个位 3×4=12 → 写2，进1
❷ 十位 0×4 + 进位1 = 1 → 写1
❸ 百位 2×4 = 8 → 写8
✨ 答案：812 ✅
🌟 小提醒
📍 中间有0，千万不能跳过！
📍 如 205×4 = 820，十位 0×4+进位的2=2，得820。
